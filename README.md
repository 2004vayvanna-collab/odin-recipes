# odin-recipes
This project is part of The Odin Project Foundations course.

The goal of this project is to build a basic recipe website using HTML. It will consist of a main index page that links to several recipe pages.

## Skills Demonstrated

By completing this project, I will have demonstrated the ability to:
- Use basic HTML structure and boilerplate
- Create and organize files and directories
- Use links and images in HTML
- Use Git for version control
- Push a project to GitHub using the command line
